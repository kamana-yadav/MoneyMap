# 💰 MoneyMap - Expense Tracker

A web-based expense tracker application built with HTML, CSS, and JavaScript.

## Features
- Add expenses with categories (Food, Travel, Bills, Shopping)
- Set monthly budget with warning alerts
- Filter expenses by category
- Data saved using localStorage
- Delete expenses

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)
